Detailed Description

Provides a class for high-performance, crash-resilient state persistence. This class encapsulates the platform-specific details of creating, mapping, and unmapping a file to memory, providing a simple, fast, and durable storage solution for an ApplicationThread's state.

An enumeration that specifies the type of a timer.

This enum class defines whether a timer should fire once (single-shot) or repeatedly (recurring).

Enumerator
SingleShot 

The timer fires once and is then disabled.

Recurring 

The timer fires repeatedly at a fixed interval.
