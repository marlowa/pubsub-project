Detailed Description

template<typename StateType>
class pubsub_itc_fw::MemoryMappedFile< StateType >

Manages a memory-mapped file for durable state persistence. This class provides a direct, low-latency interface to a file on disk by mapping it into the application's address space. It's intended for storing and retrieving the application's state to enable crash recovery.

Template Parameters
StateTypeThe type of the state to be stored.
