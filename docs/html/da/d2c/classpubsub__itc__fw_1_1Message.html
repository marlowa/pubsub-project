Detailed Description

A lightweight, non-owning wrapper for binary message payloads.

This class serves as a generic data carrier for both pubsub and inter-thread communication (ITC) messages. It holds a view (span) over raw binary data and its length. The actual data memory is managed externally (e.g., by memory pools).

This design is fully compatible with Google Protobuf. On the sending side, you serialize your Protobuf message into a buffer (from a memory pool), then create a Message that references this buffer. On the receiving side, you use the Message's data to deserialize into a Protobuf message.
