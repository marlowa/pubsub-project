Detailed Description

An expandable memory allocator composed of fixed-size pools.

This class manages a collection of FixedSizeMemoryPool instances and provides a thread-safe allocation and deallocation interface. If a pool fills up, it can allocate a new one without a set limit. It is designed to continue creating new pools until the system's memory is exhausted.
